# Spec: The Shattered Arrival

## Goals
1. Establish the **Arcane-Industrial** aesthetic through descriptive text.
2. Introduce the **"Resonance vs. Distortion"** duality via the Damaged Relic.
3. Provide the first meaningful choice that impacts player attributes (Sync, Logic, or Prowess).

## Key Deliverables
- `storylets/origin.json`: The initial storylet data for the Borderlands.
- `inventory/relic.json`: The data structure for the first Magic-Tech item.
- Narrative text for the three starting paths (Distortion, Resonance, Void).
